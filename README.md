# AuraMusic Landing Page

The official landing website for [AuraMusic](https://github.com/chila254/AuraMusic) — a modern Android music player with YouTube Music integration, powerful audio features, and a beautiful Material 3 interface.

## Tech Stack

- **Next.js 15** (React 19)
- **Tailwind CSS 4**
- **TypeScript**
- **Lucide Icons**

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

This project is configured for **Vercel** deployment. Connect the GitHub repo to Vercel and it will auto-deploy on push.
